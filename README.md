## What soundslikewhat Manager Does

soundslikewhat Manager is a browser-based HTML utility for managing a Spotify playlist through a web interface. It is designed for fast, hands-on playlist cleanup, giving you a way to inspect order, organize related tracks, and make routine edits without leaving the browser.

This tool is especially handy when you want a simple place to sort a playlist, assign sample families and roles, and catch repeated entries before pushing changes back to Spotify. The approach stays lightweight and avoids AI-driven automation, so the experience is centered on direct editing and user control.

## Core Features

- Work with a Spotify playlist directly in the browser
- Rearrange tracks with drag and drop
- Label sample families and roles for clearer organization
- Spot duplicate entries and other redundant items
- Save the revised playlist order back to Spotify
- Operate entirely in the browser with an HTML-based interface
- Keep the process manual and straightforward, with no AI-assisted steps
